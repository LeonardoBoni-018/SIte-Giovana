# Frames by Gi

Projeto React + Vite para site de apresentação de serviços de story maker.

## Como usar

1. Instale dependências:
   ```bash
   npm install
   ```
2. Rode o projeto em desenvolvimento:
   ```bash
   npm run dev
   ```
3. Faça build:
   ```bash
   npm run build
   ```

## Estrutura

- `src/pages` — páginas principais do site
- `src/components` — componentes reutilizáveis
- `src/data` — dados de serviços e portfólio
- `src/index.css` — estilos globais com Tailwind CSS

## Observação

Substitua os links de Instagram e WhatsApp pelos seus URLs reais. Adicione as imagens da logomarca e dos trabalhos em `src/assets`.
